# 🏙️ Jogo SuperTrunfo

## 🃏 Sobre o Projeto
O **SuperTrunfo** é um jogo desenvolvido em **linguagem C**. O objetivo é a comparação de atributos entre duas cidades e determinar a vencedora.

###

<h2 align="left">🏗️Estruturas Utilizadas</h2>
 <p align="left"> • if-else <br> • switch-case <br> • Operador Ternário <br> • Cálculo de Métricas</p>
  
###

<h2 align="left">📌Importância do Projeto</h2>
  <p align="left">• Aprendizado lógico em C <br> • Boas práticas de codificação <br> • Validação de entradas </p>
